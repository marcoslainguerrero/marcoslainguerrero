# 🏋️ Proyecto destacado

## TrackerGym

Aplicación web para el seguimiento de entrenamientos de gimnasio.

Permite registrar:

* Ejercicios
* Series
* Repeticiones
* Peso utilizado

Incluye **roles de cliente y entrenador**, permitiendo a los entrenadores supervisar el progreso de sus usuarios.

Tecnologías utilizadas:

* Spring Boot
* Thymeleaf
* Spring Security
* Base de datos relacional
* Arquitectura MVC
